## 用户需求

修复火山引擎文生图 API 调用失败问题。当前错误：

- API 调用返回 400 Bad Request
- 错误信息：`MissingParameter: The request is missing Action parameter`

## 问题分析

1. 当前代码直接使用 httpx 发送请求到 `https://visual.volcengineapi.com/`，但没有添加必需的 `Action` 和 `Version` 查询参数
2. 自定义签名方法不完整，不符合火山引擎签名规范
3. 火山引擎 Visual API 需要正确的 URL 路径格式：`/{Action}/{Version}/{Service}/{Method}/`

## 解决方案

使用已安装的火山引擎官方 SDK `volcenginesdkcv20240606`，该 SDK 会自动处理签名和 API 路径格式。

## 技术方案

### 使用火山引擎官方 SDK

项目已安装 `volcengine-python-sdk`，包含 `volcenginesdkcv20240606` 模块，提供：

- `CV20240606Api` - CV API 客户端
- `HighAesGeneralV13Request` - 高美感文生图请求模型
- 自动签名处理

### API 调用格式

SDK 自动构建正确的请求：

- URL: `https://visual.volcengineapi.com/HighAesGeneralV13/2024-06-06/cv/post/application_json/`
- 自动添加火山引擎签名（volcengineSign）

### 环境变量配置

需要在 `.env` 中添加火山引擎 AK/SK：

```
VOLCENGINE_ACCESS_KEY=your_access_key
VOLCENGINE_SECRET_KEY=your_secret_key
```

## 关键代码结构

```python
# 使用火山引擎 SDK 的正确方式
from volcenginesdkcv20240606 import CV20240606Api, HighAesGeneralV13Request
from volcenginesdkcore import ApiClient, Configuration

# 配置 AK/SK
configuration = Configuration()
configuration.access_key = "YOUR_AK"
configuration.secret_key = "YOUR_SK"

# 创建 API 客户端
api_client = ApiClient(configuration)
cv_api = CV20240606Api(api_client)

# 创建请求
request = HighAesGeneralV13Request(
    req_key="high_aes_general_v13",
    prompt="描述内容",
    width=1024,
    height=1024,
    return_url=True
)

# 调用 API（同步）
response = cv_api.high_aes_general_v13(request)
```
