## 用户需求

在照片预览界面和相册页面增加照片批量选择、删除功能。

## 功能概述

- 在照片画廊页面（PhotoGallery.vue）添加批量选择模式
- 用户可进入选择模式，点击照片进行多选
- 提供全选、取消选择、批量删除操作
- 选中照片时显示选中状态和数量提示

## 核心功能

1. **选择模式切换** - 进入/退出批量选择模式
2. **多选照片** - 点击照片切换选中状态
3. **全选/取消全选** - 快速选择当前页所有照片
4. **批量删除** - 一次性删除多张选中照片
5. **选中状态显示** - 照片卡片显示勾选图标

## 技术栈

- 后端：FastAPI + SQLAlchemy (异步)
- 前端：Vue 3 + TypeScript + Pinia + Tailwind CSS
- 存储：支持腾讯云COS和本地存储

## 实现方案

### 后端修改

**1. `backend/app/api/photos.py`** - 添加批量删除API

```python
@router.post("/batch-delete")
async def batch_delete_photos(photo_ids: List[str], db: AsyncSession = Depends(get_db)):
    """批量删除照片"""
    success_ids, failed_ids = await PhotoService.delete_photos_batch(db, photo_ids)
    return {"success": success_ids, "failed": failed_ids, "deleted_count": len(success_ids)}
```

**2. `backend/app/services/photo_service.py`** - 添加批量删除方法

- 复用现有的 `delete_photo` 逻辑
- 批量处理，返回成功和失败的ID列表
